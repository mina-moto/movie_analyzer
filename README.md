# MovieAnalyzer
Web Application that finds and visualizes document vectors of movie synopsis

## Environment

- node.js
- React
- python3.7
- docker

## Usage

### Dataset scraping
```shell
python filmarks_analyzer.py
```

### Calculation sentence vector
```shell
python movie_to_vec.py
```

### Builds application
```shell
./build.sh
```

### Runs application
```shell
./launch.sh
```

You can see app on localhost
